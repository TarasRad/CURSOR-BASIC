Команди, які потрібно знати:

  • git config --global user.name blabla - задати ім'я
  • git config --global user.email blabal@bla - задати електронну пошту
  • git init - ініціалігувати git-репозиторій
  • git status - перевірити стан репозиторія
  • git add - додати файли в індекс
  • git commit - закомітити зміни. Завантажити зміни на локальний репозиторій
  • git log - покаже всі коміти у даній гілці
  • git checkout - створення гілок та перехід між ними
  • git branch - показати лише локальні гілки
  • git push - завантажити з локального на віддалений репозиторій
  • git pull - завантажити з репозиторія. Отримати зміни з віддаленого репозиторія
  • git merge - об'єнання гілок
  • git clone - клонування репозиторія

Додаткові команди:

  • git revert - робить новий коміт, протилежний існуючому
  • git reset - скидання індекса робочого каталога до останнього коміта
  • git rebase - застосування комітів для іншої гілки
  • git cherry-pick - бере зміни, внесені одним комітом та застосовує до іншого
  • git - тимчасово зберігає зміни
